# 基本释义

ODWROTKA约定叫，18p+，3张以上红心。
3张红心+4张黑桃优先叫2D；
3张红心+5张黑桃优先叫1S。

# 同伴的应对

- * [[1C - 1H - 2D - 2H|2H]]：10个调整点以下，4张红心，无5张低花；
- * [[1C - 1H - 2D - 2S|2S]]：11个调整点以上，4张红心；
- * 2NT：10个调整点以下，5张红心；
- 3C：10个调整点以下，5张以上草花，4张红心；
- 3D：10个调整点以下，5张以上方块，4张红心；
- * 3H：11个调整点以上，5张红心；
- * 3S：10个调整点以下，6张红心；
- * 3NT：11个调整点以上，6张红心；
