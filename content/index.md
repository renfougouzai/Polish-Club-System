# 开叫

- [[1C]]：以下三种情况之一：  
		12-14p，均型牌，  
			存在4-4-4-1，非单张草花的极端情况；  
		15-17p，5张以上草花，非均型，  
			存在4-4-4-1，单张方块的极端情况；  
		18p+，任意牌型。
- [[1D]]：12-17p，一般保证5张方块，非均型。3种情况下仅有4张方块：  
		12-14p，5张草花+4张方块，不适合打无将；  
		12-14p，4-4-4-1，单张草花；  
		15-17p，4-4-4-1，除方块以外的某一门单张。
