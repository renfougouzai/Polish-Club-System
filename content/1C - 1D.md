# 基本释义

以下两种情况之一：
- 7p-，任意牌型，有4张高花时6p-；
- 8-11p，不符合其他叫品（一般是非均型低花套）。

# 同伴的应对

- \*[[1C - 1D - 1H|1H]]：以下四种情况之一：  
		12-14p，红心长于黑桃，  
			4-4红心黑桃（或更少的等长）先叫1H；  
		15-17p，4张以上红心，5张以上草花；  
		18-21p，5张红心；  
		18-21p，4张红心+5张以上低花；
- \*[[1C - 1D - 1S|1S]]：以下四种情况之一：  
		12-14p，黑桃长于红心，  
			5-5红心黑桃（或更多的等长）先叫1S；  
		15-17p，4张以上黑桃，5张以上草花；  
		18-21p，5张黑桃；  
		18-21p，4张黑桃+5张以上低花；
- \*1NT：18-19p，均型，可能有4张高花或5张低花；	
- 2C：15-21p，5张以上草花，非均型；  
		15-17p时没有4张高花，  
		18-21p时允许有4张高花。
- \*[[1C - 1D - 2D|2D]]：22p+，任意牌型；
- 2H：18-21p，至少6张红心好套；
- 2S：18-21p，至少6张黑桃好套；
- \*2NT：18-21p，5张方块+4张草花；
- 3C：18-21p，6张以上草花；
- 3D：18-21p，6张以上方块；
