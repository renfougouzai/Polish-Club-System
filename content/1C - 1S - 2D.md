# 基本释义

ODWROTKA约定叫，18p+，3张以上黑桃，  
3张黑桃+5张红心优先叫2H。

# 同伴的应对

- \*[[1C - 1S - 2D - 2H|2H]]：10个调整点以下，4张黑桃，无5张低花；
- \*[[1C - 1S - 2D - 2S|2S]]：11个调整点以上，4张黑桃；
- 2NT：10个调整点以下，5张黑桃；
- 3C：10个调整点以下，5张以上草花，4张黑桃；
- \*3D：10个调整点以下，5张以上方块，4张黑桃；
- \*3H：11个调整点以上，5张黑桃；
- \*3S：10个调整点以下，6张黑桃；
- \*3NT：11个调整点以上，6张黑桃；
